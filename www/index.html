Usage

$ npm install
Install node_modules.

$ bower install
Install bower_components.

$ gulp init
Init the project
Create jQuery & Foundation javascript files - Copy FontAwesome Fonts

$ gulp dev-server
Run Browser-Sync
Re-create Css and Js files at changes and reload the browser
